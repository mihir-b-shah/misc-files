The Growl Programming Language

Growl is a general-purpose language similar to C.

Grammar

Primitive Types

  • Char: A signed 8-bit integer. Also useful for ASCII storage.
  • uChar: An unsigned 8-bit integer.
  • Int: A signed 32-bit integer.
  • uInt: An unsigned 32-bit integer.
  • Long: A signed 64-bit integer.
  • uLong: An unsigned 64-bit integer.
  • Bool: A boolean, 8-bit value.
  • Float: A 32-bit floating point number.
  • Double: A 64-bit floating point number.
